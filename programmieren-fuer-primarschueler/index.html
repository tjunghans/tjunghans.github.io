Programieren für Primarschüler

JavaScript

Die beliebteste Programmiersprache der Welt ist JavaScript. Sie ist die Programmiersprache, welche im Internet am häufigsten verwendet wird. Damit du selbst JavaScript programmieren kannst, brauchst du nur einen PC oder Mac, einen Webbrowser und einen Text-Editor.

Programmiersprache
Eine Programmiersprache ist eine Sprache, die ein Computer versteht. Wie Menschen, können Computer verschiedene Sprachen verstehen. Man nennt sie Programmiersprachen, weil man mit den Sprachen Programme die auf dem Computer gespeichert sind starten kann. Man kann auch neue Programme erstellen.
JavaScript
JavaScript ist eine Programmiersprache, welche der Webbrowser verstehen kann.
Webbrowser
Mit dem Webbrowser kann man im Internet surfen. Zu den beliebteste Webbrowser gehören Firefox, Edge, Safari und Chrome. Es spielt keine Rolle, welchen dieser Browser du für diese Übung verwendest.
